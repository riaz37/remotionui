# Component Atlas

RemotionUI organizes components by **motion role**, not UI widget type. Each item belongs to an atlas lane and is tagged by what drives its animation.

## Lanes

| Lane | What it is |
|------|------------|
| **Atoms** | Frame-level enter/exit motion |
| **Signals** | Data- and media-driven motion (captions, audiograms) |
| **Vectors** | SVG paths, shapes, draw-on reveals |
| **Spatial** | Maps, camera moves, geospatial motion |
| **Blocks** | Composed scene layouts |
| **Cuts** | TransitionSeries presets |
| **Reels** | Full composition templates |

## Tiers

- **Core**: only requires `remotion`
- **Advanced**: installs additional `@remotion/*` or map packages via the CLI
