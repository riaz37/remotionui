RemotionUI lets you add production-ready Remotion video components directly into your project. No black-box npm package — you own the code.

## Quick start

```bash
npx remotion-ui@latest init my-video
cd my-video
npx remotion-ui@latest add fade-in
```

## Philosophy

- **Registry-first** — components are distributed as source via JSON registry
- **Copy-paste, not import** — files land in your `src/remotion/` directory
- **CLI only on npm** — the `remotion-ui` package is the installer, not the components
