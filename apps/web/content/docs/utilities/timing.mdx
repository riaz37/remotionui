<InstallCommand name="timing" />

```ts
import {
  secondsToFrames,
  staggerDelay,
  enterProgress,
  exitProgress,
  EASING_ENTER,
} from "@/remotion/lib/timing";

const delay = staggerDelay(index, 8);
const opacity = enterProgress(frame, delay, 30);
```

Use `enterProgress` for entrances and `exitProgress` for exits. Most primitives depend on this lib.
